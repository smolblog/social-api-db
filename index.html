The Social API Database#

A place to learn about and document the new ways for social media sites to communicate. Whether ActivityPub, Webmention, or something entirely different, we aim to show in plain terms what developers can expect both in terms of spec compliance and what a popular project will actually send.

Any and all help is appreciated, and you can get involved via the site GitHub project.
